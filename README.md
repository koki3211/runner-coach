# Runner Coach

A mobile-first running training planner inspired by Apple HIG design principles.

## Features

- **Today View** — Today's workout details with progress tracking
- **Training Plan** — Week-by-week schedule with completion checkmarks
- **Friends** — See your running buddies' training & streaks
- **Goal Setup** — Pick a race and target time to generate a plan

## Getting Started

Open `index.html` in a mobile browser (or use Chrome DevTools device mode).

## Design Tokens

See `design-tokens/` for the variable system:
- `variables.css` — Colors, spacing, typography, shadows (light + dark mode)
- `font-family.css` — Apple system font stacks
